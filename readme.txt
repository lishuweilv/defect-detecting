环境pyhon3
图像识别的基本算法和demo
./canny                     实现canny算子检测边缘
./color_differeent          分离图片的RGB通道，用颜色对图片进行区分
./data                      生成或者用到的一些图片资源
./Harris                    角点检测
./moubanpipei               模板匹配
    ./draw_scores.py        对多角度模板(大mi  logo)进行匹配，求相关图
    ./drew_xiaozi.py        对小字进行多角度模板匹配，求相关图
./threshholding             对图片进行二值化，可以用滑板调节二值化的值
    ./change_num.py         找大milogo中的二值化阈值
    ./xiaozi.py             找小字二值化的阈值
./recognize                 识别图片中的文字
./CropPhone                 识别图片中的手机，裁剪，矫正
./xiaozi_chuli              对图片进行开运算和闭运算，处理图片
./Cut_pic                   裁剪图片制定区域
    ./demo.py               裁剪大mi字loga
    ./cut_2.py              裁剪小字的logo
./np_list.py                绘制图片的直方图
